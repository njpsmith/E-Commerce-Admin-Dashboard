# Products

## Purpose

Manage products in the e-commerce admin dashboard: list, filter, and view products.
Accessible to all authenticated users, regardless of role. The table has pagination and can be filtered.

## API endpoints

`GET /api/products` – list with filters + pagination
`GET /api/products/:id` – detail
